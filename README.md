# RAG-Implementation-DEMO

## Overview
A simplified Retrieval-Augmented Generation (RAG) system using:
- FLAN-T5 (generator)
- SentenceTransformer (retriever)
- JSON knowledge base

## How to Run
pip install -r requirements.txt
python app.py

Open: http://127.0.0.1:5000

## Notes
- This is a simplified implementation (no Top-K, small dataset)
- Demonstrates core idea of combining retrieval + generation
